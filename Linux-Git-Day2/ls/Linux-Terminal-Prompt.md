Normally, the prompt in the terminal looks like:
```bash
	<username>@<hostname><current working directory>$
```
The $ stands for a normal users who is not a root/admin

When the current working directory is home directory for the given user, it looks like:
```bash
	<username>@<hostname>[~]$
```
For a root user with admin priveleges:
```bash
	root@hostname[~]#
```

Note: It's not specifically like this but is similar as in username, then hostname, then working directory followed by $ or # to indicate user priveleges.
