
INSTALLING:
1. Run PowerShell as administrator
2. Run the command below:
```bash
	wsl --install
```
3. Restart the machine


RUNNING:
- You could open wsl by using start menu search and searching wsl and opening.
- The other way (preffered way) is to navigate the file system in powershell and when you are at the suitable folders, just type 'wsl' into the terminal and click enter.
