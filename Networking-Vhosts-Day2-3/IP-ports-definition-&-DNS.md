IP Address is a numerical label representing a device connected to the network. It looks like xxx.xxx.xxx.xxx

It is important because the routers uses IP(Internet Protocol) for communication. In other words, IP addresses are used to find the optimal path for data to reach from starting point to indended destination between interconnected networks.

IP addresses are either ipv4 which is a 32-bit address space or ipv6 which are formatted in eight groups of four hexadecimal digits, an example being 2001:0db8:85a3:0000:0000:8a2e:0370:7334.



Port is a number assigned to a specific process or service on the network. This allows devices to sort and direct network traffic correctly.

Examples are how 0-1023 are well-known ports. For example, FTP would use port 20 and 21, HTTP 80, and HTTPS 443.
1024-49151 are Registered ports which are assigned by IANA (Internet Assigned Numbers Authority)
49152-65535 are ephemeral ports which are often used by clients when they are talking to servers on the internet.



DNS (Domain Name System) where it helps us find the IP address of a domain name. This means that we get to write the domain name instead of the IP Address thanks to DNS.
